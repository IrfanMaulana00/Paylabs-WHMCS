== Installation ==

1. Download plugin Paylabs WHMCS
2. Masuk ke file manager hosting Anda lalu ke folder root WHMCS Anda
3. Upload file plugin yang sebelumnya telah didownload
4. Setelah selesai upload, Extract file plugin tersebut
5. Setelah proses extract selesai, log in ke dalam admin WHMCS anda.
6. Pada halaman admin WHMCS anda, pilih pengaturan di kanan atas > Apps & Integrations
7. Pilih tab Browse, lalu pilih Payments
8. Scroll ke bagian bawah dan cari Paylabs Payment Gateway
9. Klik dan pilih activate, lalu klik manage
10. Isi semua field sesuai data yang diberika oleh paylabs
11. klik Save Changes
12. Plugin Paylabs Payment Gateway siap digunakan

For whmcs documentation you may see here: https://developers.whmcs.com/payment-gateways/installation-activation/
Install & activate the plugin. Modify some options in the settings page to suits your need.
